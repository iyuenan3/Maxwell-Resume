## 📥 使用方法

### 查看/打印简历
1. 在浏览器中打开 HTML 文件
2. Ctrl+P（或 Cmd+P）打印
3. 选择"另存为 PDF"
4. 确认 A4 纸张，2 页

### 下载
```bash
# 克隆仓库
git clone https://github.com/iyuenan3/Maxwell-Resume.git
cd Maxwell-Resume
```
